# 北京医院智能推荐 - MVP 版本

## 📋 项目说明

基于 Gemini 3 生成的前端设计 + 我们收集的北京三甲医院数据，实现的医院智能推荐 H5 页面。

## ✨ 功能特点

### 1. 智能搜索
- 用户输入症状（如：嗓子疼、骨折、头疼）
- AI 分析推荐科室
- 筛选该科室排名前10的医院

### 2. 医院展示
- 15家北京顶级三甲医院
- 医院基本信息
- 优势科室及全国排名
- 特色科室标签
- 一键拨打电话

### 3. 用户体验
- 移动端优先设计
- 流畅的动画效果
- 快捷搜索标签
- 底部导航栏

## 🏗️ 技术栈

- **前端框架**：原生 HTML + JavaScript
- **样式**：Tailwind CSS
- **图标**：Lucide Icons
- **字体**：Noto Sans SC

## 🚀 使用方法

### 本地预览
```bash
# 进入前端目录
cd /root/.openclaw/workspace-collector/projects/hospital-finder/frontend

# 使用 Python 启动本地服务器
python3 -m http.server 8000

# 或使用 Node.js
npx serve .
```

然后访问：http://localhost:8000

### 部署到 Vercel
```bash
# 安装 Vercel CLI
npm i -g vercel

# 部署
cd frontend
vercel
```
